Keyboard Shortcuts

You can configure Keyboard Shortcuts on the following chrome scheme URI.
Due to Google Chrome security restriction, you need to copy and paste the following link to a new window (tab) to change the shortcut settings:

chrome://extensions/shortcuts

This is a fork version of original Google Chrome Extention for Pinboard.
